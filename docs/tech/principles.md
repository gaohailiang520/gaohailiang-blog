# 《原则》读书笔记

> **作者**：Ray Dalio（瑞·达利欧）
> **推荐理由**：桥水基金创始人的人生和工作原则体系，我的年度目标检查框架就来源于此

---

## 一、核心框架：五步流程

达利欧认为，任何目标都可以通过以下五步达成：

```
1. 目标 → 2. 问题 → 3. 诊断 → 4. 方案 → 5. 执行
```

### Step 1：设定清晰的目标

- 目标要**具体可量化**，不能模糊
- 不要混淆"目标"和"欲望"——目标是你真正想要的东西，欲望会阻碍你
- **排列优先级**：你不能拥有一切，选择最重要的 1-3 个目标
- 我的应用：健身 1 万粉、内源 1 万下载、量化年化 20%

### Step 2：找到阻碍目标的问题

- **对问题不回避**——这是大多数人失败的地方
- 问题分两类：技能问题和习惯问题
- 不要容忍问题，每发现一个问题就是找到了一处可改进的点
- 我的问题：时间不够、意志力不足、外部阻力

### Step 3：诊断根本原因

- 问至少 5 个"为什么"，直到找到根本原因
- 区分** proximate cause（近因）**和** root cause（根本原因）**
- 根本原因通常只有一个，不要用战术勤奋掩盖战略懒惰
- 人的问题通常比技术问题更重要

### Step 4：设计解决方案

- 方案要**具体到可执行的步骤**
- 考虑多种路径，不要一条道走到黑
- 把方案设计成**系统**而不是靠意志力
- 我的方案：固定时段做固定的事、降低任务粒度到 5 分钟

### Step 5：坚决执行

- 执行的关键是**习惯和环境**，不是意志力
- 建立问责机制——让别人知道你的计划
- 连续 7 天没执行 → 目标设太高，降低标准先动起来

## 二、两大原则分类

### 生活原则

| 原则 | 核心思想 |
|------|---------|
| **拥抱现实，应对现实** | 看清真相比希望真相是什么更重要 |
| **极度开放** | 你的大脑有两大障碍：ego（自我意识）和 blind spots（盲点） |
| **从自然的角度看事物** | 进化是生命最大的驱动力 |
| **痛苦 + 反思 = 进步** | 痛苦是进化的信号，不是惩罚 |
| **5 步流程** | 见上文，这是全书最精华的部分 |

### 工作原则（桥水基金实践）

| 原则 | 核心思想 |
|------|---------|
| **极度求真** | 不掩饰问题，当面说真话 |
| **创意择优** | 最好的想法胜出，不管来自谁 |
| **可信度加权** | 对多次成功的人的意见给予更高权重 |
| **把问题摆在桌面上** | 不要背后议论，要当面讨论问题 |
| **系统化决策** | 把决策原则写下来，形成系统 |

> 工作原则本质上是生活原则在组织中的应用

## 三、极度开放：全书最重要的原则

达利欧说，阻碍你成功的最大障碍是两个：

### 1. Ego（自我意识）

```
你的大脑结构决定了：
  - 前额叶皮层（理性） → 最晚进化出来，最弱
  - 杏仁核（情绪/防御） → 最早进化出来，最强

当你被质疑时，杏仁核会触发"战斗或逃跑"反应
这就是为什么你会在争论中变得情绪化
```

**解法**：意识到你的"两个你"——那个想争辩的你和那个想学习的你。让后者主导。

### 2. Blind spots（盲点）

> 每个人都有自己看不到的维度。你觉得自己是对的，只是因为你的视角有限。

- 你无法看到自己看不到的东西
- 这就是为什么你需要别人的视角
- 极度开放 = 主动寻找和你观点不同的人

### 练习极度开放的方法

1. **每次被批评时**，先深呼吸 3 秒，然后说"谢谢，能详细说说吗？"
2. **每次争论时**，先复述对方的观点让对方确认"你说得对，我就是这个意思"
3. **做决策前**，找 2-3 个和你观点不同的人聊一聊
4. **记录你的盲点**——什么样的反馈让你最不舒服？那可能就是你的最大盲点

## 四、痛苦 + 反思 = 进步

这是达利欧最重要的公式：

```
痛苦 → 你的系统在告诉你"这里需要进化"
  ↓
如果你回避痛苦 → 你错过了一次进化机会
  ↓
如果你反思痛苦 → 你找到根源 → 你进化了 → 下次同样的痛苦不来了
```

### 如何应用

**工作中遇到难搞的 Bug？**
- 痛苦 ← 解决不了
- 反思 ← 是知识盲区？方法不对？工具不熟？
- 进步 ← 补上这个短板，以后这类问题不再卡住

**健身坚持不下去？**
- 痛苦 ← 不想去健身房
- 反思 ← 是计划太激进？时间不合适？恢复不够？
- 进步 ← 调整计划，形成可持续的习惯

> 痛苦不是惩罚，是你在进化的信号。

## 五、可信度加权（Believability Weighting）

达利欧在桥水实行的决策机制：

```
不是所有人意见平等，而是：
  多次在相关领域成功的人 → 意见权重高
  没有相关经验的人 → 意见权重低
```

**但有一个关键前提**：你必须对别人有**极度透明的记录**，才能知道谁可信。

### 对我有什么用？

- 做重要决策时，**征求有结果的人的意见**
- 在健身领域，听有训练痕迹的人的建议
- 在量化领域，听有实盘经验的人的建议
- 不要向从未做过这件事的人请教怎么做

---

## 六、把原则系统化

达利欧的核心思想之一是：**把原则写下来，形成决策系统。**

- 每次做决策时，记录下来：我当时依据什么原则？
- 决策后复盘：这个原则对吗？
- 不对就修改原则，形成迭代

### 我的做法

```
读完这本书后，我把五步法写进了 README 附录
每次复盘时对照检查：
  □ 目标是否清晰？
  □ 问题是否诚实面对？
  □ 诊断是否找到了根本原因？
  □ 方案是否可执行？
  □ 是否在持续执行？
```

## 七、总结

| 核心理念 | 一句话 |
|---------|--------|
| 五步流程 | 目标 → 问题 → 诊断 → 方案 → 执行 |
| 极度开放 | 你的观点可能是错的，主动找人挑战你 |
| 痛苦+反思 | 痛苦不是坏事，是进化信号 |
| 可信度加权 | 听有结果的人的意见 |
| 系统化 | 把原则写下来，不断迭代 |

1. **先读第三章**（五步流程）——这是最实用的部分

3. **读生活原则部分**（第二大部分）
4. **最后读工作原则**（第三大部分）——如果你管理团队
