# 《高效能人士的七个习惯》读书笔记

> **作者**：Stephen R. Covey（史蒂芬·柯维）
> **副标题**：30 周年纪念版 — 全球销量超 4000 万册
> **推荐理由**：这不是一本时间管理书，而是一本关于「如何成为一个成熟的人」的书

---

## 一、七个习惯总览

柯维把七个习惯分为三个阶段，从依赖到独立再到互赖：

```
依赖期 → 独立期（个人领域的成功）→ 互赖期（公众领域的成功）
                       ↓
                  不断更新（持续进化）
```

| 阶段 | 习惯 | 核心 |
|------|------|------|
| **个人领域**（独立） | ① 积极主动 | 你是创造者，不是受害者 |
| | ② 以终为始 | 先想清楚墓志铭上写什么 |
| | ③ 要事第一 | 重要的事先做，别被紧急的事绑架 |
| **公众领域**（互赖） | ④ 双赢思维 | 不是非输即赢，可以双赢 |
| | ⑤ 知彼解己 | 先听懂别人，再让别人听懂你 |
| | ⑥ 统合综效 | 1+1 > 2 的创造力 |
| **持续进化** | ⑦ 不断更新 | 在身体、心智、社交、精神四个维度投资自己 |

## 二、习惯 1：积极主动

### 核心思想

> **刺激与回应之间，存在一个空间。在这个空间里，是我们的自由和力量。** — 维克多·弗兰克尔

```
外部刺激 → [选择的空间] → 回应
```

大多数人的反应模式是：
- 别人激怒你 → 你生气（刺激-反应，没有间隔）
- 环境不好 → 你抱怨（被动反应）

积极主动的人：
- 别人激怒你 → **停下 3 秒** → 选择回应方式
- 环境不好 → 问"我能做什么？"

### 关注圈 vs 影响圈

```
        ┌──────────────┐
        │   关注圈      │  （你关心的事）
        │  ┌──────────┐ │
        │  │  影响圈   │ │  （你能改变的事）
        │  │          │ │
        │  └──────────┘ │
        └──────────────┘
```

**积极主动的人**把精力放在影响圈（我能做什么？）
**消极被动的人**把精力放在关注圈（都是别人的问题！）

### 我的应用

- **健身**：不抱怨工作忙没时间 → 固定 6 点起床，先把训练做了
- **内源项目**：不抱怨公司不支持 → 先做出 MVP，让结果说话
- **量化**：不抱怨市场不好 → 研究策略，控制风险

## 三、习惯 2：以终为始

### 核心思想

> 在做任何事之前，先想清楚"终点"是什么。

**最有力量的练习**：想象你在自己的葬礼上，你希望别人怎么评价你？

- 家人说：他是一个有爱的人
- 同事说：他是一个靠谱的人
- 朋友说：他是一个值得信任的人

反过来推：**你现在应该怎么做？**

### 写作个人使命宣言

一个好的使命宣言应该回答：
1. 我**最看重**什么？（价值观）
2. 我想成为**什么样的人**？（身份）
3. 我想留下**什么**？（影响）

#### 我的使命宣言

```
我是一个「掌控型」的人：
  - 我能掌控自己的身体（健身）
  - 我能掌控自己的技术（内源项目 + 驱动）
  - 我能掌控自己的财富（量化）

我对自己的要求：
  - 说到做到，不找借口
  - 持续学习，不断进化
  - 对结果负责，不抱怨环境
```

### 以终为始的检查清单

- [ ] 你正在做的事情，和你的终极目标一致吗？
- [ ] 如果你只剩一年生命，你还会做这件事吗？
- [ ] 你今天的努力，是在为那个"葬礼上的评价"积累吗？

## 四、习惯 3：要事第一

### 时间管理四象限

```
                   紧急性
                ┌─────┬─────┐
                │ 紧急  │ 不紧急│
          ┌────┼─────┼─────┤
   重要性 │ 重要 │ Ⅰ    │ Ⅱ    │
          │     │危机   │要事   │
          │     │截止   │锻炼   │
          │     │      │学习   │
          ├────┼─────┼─────┤
          │不重要│ Ⅲ    │ Ⅳ    │
          │     │干扰   │浪费时间│
          │     │琐事   │刷手机  │
          └────┴─────┴─────┘
```

**关键洞察**：
- 大多数人被困在 **象限 I**（紧急重要）——一直在救火
- 高效能人士把大部分时间花在 **象限 II**（重要不紧急）——这才是杠杆

### 我的时间分配按象限

| 象限 | 活动 | 每周时间 |
|------|------|---------|
| Ⅱ（重要不紧急） | 健身、内源项目开发、量化策略研究、驱动学习 | 25-30h |
| Ⅰ（紧急重要） | 工作 Deadline、紧急 Bug | 不固定 |
| Ⅲ（紧急不重要） | 无效会议、被拉去救火 | 尽量压缩 |
| Ⅳ（不紧急不重要） | 刷短视频、无目的浏览 | 0 |

### 如何做到要事第一

1. **每周计划**：先放象限 II 的事（健身时间块、深度工作时间块）
2. **把大石头先放进去**：如果这周只能做 3 件事，是哪 3 件？
3. **说不**：对象限 III 和 IV 的事情果断说不
4. **固定时段**：每天 6:00-7:00 健身，20:00-21:00 学习/写代码

## 五、习惯 4：双赢思维

### 六种人际交往模式

```
┌─────────────────────────────────┐
│  双赢（Win-Win）      ✓ 最好的  │
│  我赢你输（Win-Lose）    大多数   │
│  我输你赢（Lose-Win）    老好人   │
│  双输（Lose-Lose）       报复    │
│  我赢（Win）            自私     │
│  不合作（No Deal）   ✓ 好选择   │
└─────────────────────────────────┘
```

### 双赢的核心不是让步，是找到第三选择

> 双赢不是"你退一步我退一步"，而是一起找到**让双方都更满意的方案**。

**工作中应用**：
- 领导要你加班做 A 项目 → 你想做 B 项目
- 双输：你不加班，领导不满意
- 妥协：你加班但心里不爽
- 双赢：你主做 A 但把 B 的探索时间纳入 A 的项目计划

### No Deal 也是好选择

> 有时候最好的双赢就是**不合作**。

- 价值观不合 → 不合作比勉强合作好
- 条件谈不拢 → 坦诚说"这次不行"比强行合作好

## 六、习惯 5：知彼解己

### 核心思想

> 大多数人不是为了理解而倾听，而是为了回应而倾听。

**你以为你在倾听，实际上你在准备反驳。**

### 倾听的四个层次

| 层次 | 描述 | 效果 |
|------|------|------|
| 忽视 | 不听 | 对方觉得你不尊重 |
| 假装 | 点头嗯嗯 | 对方觉得你在敷衍 |
| 选择性听 | 只听你想听的 | 你误解了对方 |
| **共情倾听** | 试着理解对方的视角和感受 | 真正的沟通开始 |

### 共情倾听的练习

当你和别人有分歧时，试试这个流程：

1. **"你的意思是……"** — 复述对方的核心观点
2. **"我理解你的感受是……"** — 说出对方的情绪
3. **"对吗？"** — 让对方确认你的理解正确
4. **"我的角度是……"** — 然后再表达你的观点

> 关键规则：**在对方觉得被理解之前，不要试图说服对方。**

### 解己（让别人理解你）

> 你要先理解别人，然后才能被别人理解。

- 用对方的语言说话
- 用故事和类比而不是抽象概念
- 逻辑 + 情感（人们做决定靠情感，用逻辑辩护）

## 七、习惯 6：统合综效

### 核心思想

> 1 + 1 > 2 — 当两个不同视角的人真诚合作时，能产生第三种方案，比任何一方的方案都好。

### 为什么大多数团队做不到统合综效

```
信任度越低 → 防御越多 → 创意越少 → 统合综效不可能
```

| 信任水平 | 沟通模式 | 结果 |
|---------|---------|------|
| 低 | 防御/妥协 | 1+1=1.5 |
| 中 | 尊重/折中 | 1+1=2 |
| 高 | 创造性合作 | 1+1≥3 |

### 如何实现统合综效

1. **建立安全环境**——"这里可以提反对意见"
2. **拥抱差异**——不同视角是资产，不是障碍
3. **第三方案**——不是你的方案也不是我的，是更好的方案

### 在目标中的应用

- **健身 + 内源**：健身素材剪辑过程中，可以练习视频编辑技能，反哺内源项目的多媒体能力
- **驱动 + 内源**：驱动开发中遇到的问题和解决方案，可以写成博客，充实内源项目的内容
- **量化 + 原则**：量化交易的回测本质上就是在验证你的"交易原则"是否有效

## 八、习惯 7：不断更新

### 四个维度的投资

```
        ┌──────────────┐
        │   身体维度    │
        │  （健身）     │
        ├──────────────┤
        │   心智维度    │
        │  （阅读/写作） │
        ├──────────────┤
        │   社交维度    │
        │  （关系/连接） │
        ├──────────────┤
        │   精神维度    │
        │  （价值观/使命）│
        └──────────────┘
```

### 我的四个维度投资

| 维度 | 我的做法 | 频率 |
|------|---------|------|
| **身体** | 力量训练（深蹲/硬拉/卧推） | 每周 4-5 次 |
| **心智** | 读技术书 + 写博客 + 读论文 | 每天 1 小时 |
| **社交** | 内源项目用户沟通 + 健身内容互动 | 每周分散 |
| **精神** | 复盘目标 + 阅读使命宣言 | 每周日 |

### 螺旋上升

七个习惯不是一次性学会的，而是螺旋上升的：

```
        更成熟 ← 习惯 7 不断更新
          ↑
        互赖 ← 习惯 4、5、6
          ↑
        独立 ← 习惯 1、2、3
          ↑
        依赖（起点）
```

每经过一轮，你的成熟度和效能就上升一个层次。

## 九、总结

### 七个习惯对照表

| 习惯 | 一句话 | 对我的意义 |
|------|--------|-----------|
| ① 积极主动 | 关注影响圈 | 不抱怨，问"我能做什么" |
| ② 以终为始 | 先写墓志铭 | 写个人使命宣言，明确方向 |
| ③ 要事第一 | 先放大石头 | 每周先排健身和学习时间 |
| ④ 双赢思维 | 找第三选择 | 合作不是零和游戏 |
| ⑤ 知彼解己 | 先听再被听 | 先理解对方，再争取被理解 |
| ⑥ 统合综效 | 1+1>3 | 拥抱差异，创造更好的方案 |
| ⑦ 不断更新 | 投资四个维度 | 身体+心智+社交+精神 |

### 这本书和其他书的关系

```
《原则》           → 教你系统化的决策框架
《高效能人士的七个习惯》 → 教你成为一个成熟的人
《Atomic Habits》  → 教你如何执行（习惯系统）
```

三本书合在一起 = **方向正确（原则）+ 心智成熟（习惯）+ 执行到位（习惯系统）**
